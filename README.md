# مخزون الصيدلية — Pharmacy Inventory Management

نظام إدارة مخزون الصيدلية لمستشفى الشطرة العام — PWA على Firebase

- **النسخة**: 7.5
- **الإنتاج**: https://phzain14.web.app
- **Firebase Project**: phzain14
- **نظام التشغيل**: Online-Only PWA
- **التقنيات**: Vanilla JS + Firebase + Chart.js + jsPDF

## النشر

النشر يدوي عبر GitHub Actions:
1. Actions → "🚀 نشر إلى الإنتاج"
2. Run workflow → target = `hosting` (أو `all`) → confirm = `نشر`

راجع `DEPLOYMENT.md` للتفاصيل الكاملة.
